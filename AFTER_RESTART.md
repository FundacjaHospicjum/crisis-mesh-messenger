## 🐛 If Something Goes Wrong

### "flutter: command not found"
Your computer might need one more restart for PATH to fully update.

### "No devices available"
Run: `flutter config --enable-windows-desktop`

### Build errors
Run:
```powershell
flutter clean
flutter pub get
flutter run -d windows
```
